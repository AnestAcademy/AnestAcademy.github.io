KHÓA HỌC JAVA WEB CƠ BẢN.

Cấp độ

CƠ BẢN

Thể loại

Web

Thời lượng

25 ~ 30 giờ

Thông tin

– Khóa học này sẽ cung cấp cho bạn những kiến thức - khái niệm mới về Java Web, giúp bạn làm quen và viết code với Servlet và JSP.
– Sau khoá học bạn sẽ có thể code được một chương trình Java web đơn giản với JSP và Servlet, có thể tự tin viết code, làm project trên lớp hoặc lab Java Web một cách đơn giản.

Đối tượng

– Những bạn đã có kiến thức cơ bản về Java Core + HTML + Database.
– Là khóa học dành cho những bạn mong muốn tìm một nơi học - khoá học chất lượng cao để học thật - lấy kiến thức thật, mang tính thực tiễn có thể áp dụng vào công.
– Dành cho những bạn muốn lập trình một website đơn giản cho mục đích cá nhân với qui mô vừa và nhỏ.
– Dành cho những bạn mong muốn làm quen - học hỏi với những người có kinh nghiệm trong lĩnh vực lập trình.

Kiến thức nhận được

– Nắm chắc kiến thức về Java JDBC.
– Hiểu được Java Web là gì - có thể làm được gì?
– Nắm được kiến thức về JSP và Servlet.
– Nắm được cách sử dụng Session và Cookie.
– Nắm được cách thiết kế một chương trình web đơn giản theo mô hình MVC.
– Làm project cuối khóa theo chuẩn và kết hợp những thứ đã học vào một Project duy nhất.
– Định hướng cách học tốt nhất và hiệu quả khi theo con đường làm lập trình viên.
