Thông tin

– Khóa học này sẽ cung cấp cho bạn những khái niệm cốt lõi và cơ bản nhất của Java OOP, giúp bạn làm quen và viết code một cách thành thạo ngôn ngữ Java.
– Sau khoá học bạn sẽ hiểu rõ bản chất của Java OOP cơ bản, có thể tự tin viết code, làm Project trên lớp hoặc Lab Java Core một cách đơn giản.
– Hiện tại Java vẫn là ngôn ngữ được ưa chuộng và được nhiều nơi tuyển dụng nhất. Học chắc Java Core sẽ tạo cho bạn một nền tảng vững chắc để học tiếp ngôn ngữ Java lên cao hơn trong tương lai.

Đối tượng

– Những bạn sắp lên học chuyên ngành kỳ 2 hoặc đang học kỳ 2 - đã học xong lập trình cơ bản với ngôn ngữ C.
– Là khóa học dành cho những bạn mong muốn tìm một nơi học - khoá học chất lượng cao, có sự hỗ trợ nhiệt tình, mang lại những kiến thức thật - thực tiễn và bổ ích nhất.
– Dành cho những bạn đang không vững - mất gốc về Java, đang khó khăn với Lab Java Core, Java Desk, Java Web vì hổng kiến thức Core, muốn học lại để hiểu rõ và nắm chắc bản chất.
– Những bạn mong muốn làm quen - học hỏi với những người có kinh nghiệm, cựu sinh viên trong lĩnh vực lập trình.

Kiến thức nhận được

– Hiểu được Java OOP là ngôn ngữ như thế nào - có thể làm được gì.
– Nắm được những tính chất đặc trưng của ngôn ngữ Java - lập trình hướng đối tượng là gì?
– Những đặc tính ấy được thể hiện trong code và khi thực hiện code như thế nào?
– Học được cách phân tích vấn đề và giải quyết vấn đề trong lập trình.
– Làm project cuối khóa theo chuẩn và kết hợp những thứ đã học vào một project duy nhất.
– Định hướng cách học tốt nhất và hiệu quả trong quá trình học lập trình.

Q & A

Những câu hỏi thường gặp. Bạn hỏi - chúng tôi trả lời.

Có nên lựa chọn học Java?
Có. Hiện nay Java vẫn rất phổ biến và mạnh mẽ, ở Việt Nam vẫn đang có rất nhiều công ty tuyển dụng lập trình viên Java với mức lương tương đối cao.
Học một mình ngôn ngữ Java là đủ?
Java rất rộng và là một ngôn ngữ bậc cao, giải quyết được rất nhiều bài toán trong thực tế. Nhưng để có thể làm trọn vẹn và giúp bạn sau này có thể tiến xa hơn thì cần phải học thêm vài ngôn ngữ lập trình khác nữa. Ví dụ theo đường lập trình Java Web thì cần học thêm HTML/CSS, Javascript...
Khi có câu hỏi hoặc thắc mắc liên quan đến bài học - tôi sẽ được hỗ trợ qua kênh nào?
Bạn sẽ được hỗ trợ 24/7 qua các kênh online như nhóm facebook, inbox trực tiếp facebook cá nhân Mentor... hoặc được hỗ trợ trực tiếp tại trung tâm.
Các bạn sẽ được support về việc học qua group không chỉ là môn đang học mà còn các môn lập trình khác.
Tiền học phí sẽ đóng khi nào?
Học phí sẽ được thu trực tiếp trong 3 buổi đầu hoặc nhận chuyển khoản trước, hoàn trả 100% nếu cuối khóa bạn cảm thấy không hài lòng. Bạn có thể đóng trước lúc đăng ký để giữ chỗ, ưu tiên xếp lớp cho ai đóng học phí trước và chắc chắn đi học.
Nếu trong quá trình học mà có việc bận phải nghỉ giữa chừng thì sao?
Bạn sẽ được hoàn trả lại học phí - chỉ thu số tiền ứng với số buổi bạn đã học.
